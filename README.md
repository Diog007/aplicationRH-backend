# 🏥 Application - RH

Aplicação web em Java que simplifica o processo de cadastro de vagas de trabalho. Esta plataforma oferece método para listar todas as vagas disponíveis e permite o cadastro fácil e rápido de candidatos para essas vagas.

## 🛠️ Compilação
Para gerar os arquivos-fonte, siga o padrão de um projeto Maven: `mvn clean install`.

Aqui estão alguns links úteis para a aplicação local:
- Listar Vagas: http://localhost:8080/vagas
- Cadastrar Vagas: http://localhost:8080/cadastrarVaga
- Detalhar um Vaga: http://localhost:8080/1

## 📚 Como começar
Existem duas maneiras principais de começar com o Application RH:
1. [Baixe o ZIP do projeto](https://github.com/Diog007/aplicationRH-backend/archive/refs/heads/main.zip) e abra-o em uma IDE de sua escolha.
2. Clone o repositório usando o seguinte comando: `https://github.com/Diog007/aplicationRH-backend.git`.

Depois de obter o código-fonte, você pode instalar as dependências do projeto com `mvn clean install` e iniciar a aplicação com `mvn spring-boot:run`. A aplicação estará disponível na porta 8080. Para alterar a porta, modifique o `server.port` no arquivo `application.properties`.


## ⚙️ Funcionalidades

- **Cadastro de Vagas:** Registre vagas de trabalho de forma rápida e simples, inserindo todas as informações relevantes.

- **Atualização de Vagas:** Mantenha as vagas de trabalho sempre atualizadas, permitindo realizar alterações conforme necessário.

- **Listagem de Vagas :** Encontre todas as vagas disponíveis de maneira simples e organizada, facilitando a busca por candidatos e a visualização das oportunidades.

- **Cadastro de candidato:** Cadastre candidatos para vagas específicas de maneira eficaz e simples, permitindo registrar suas informações pessoais
